## 🛠️ Troubleshooting

### The app will not open
- Make sure you extracted the ZIP file first
- Right-click the app and choose Run as administrator
- Check that Windows did not block the file

### Sign-in does not finish
- Close the browser and try again
- Make sure you are signed in to the right Google account
- Allow pop-ups in your browser

### Files do not show up
- Refresh the app
- Check your internet connection
- Confirm that the account has access to the Drive files

### Uploads fail
- Check file size
- Make sure the file name does not use strange symbols
- Try again on a stable connection

### The folder list looks empty
- Open the correct Google account
- Wait for sync to finish
- Refresh the view in the app
